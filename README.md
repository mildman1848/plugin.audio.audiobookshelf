# plugin.audio.audiobookshelf

Deutsch: [README.DE.md](README.DE.md)

Kodi addon for Audiobookshelf with login support, playback, progress sync, and STRM export for
audiobooks and podcasts.

## Repository Structure

- `plugin.audio.audiobookshelf/`: installable Kodi addon
- `plugin.audio.audiobookshelf/resources/lib/`: API, player, and helper logic
- `scripts/build_release.py`: builds release ZIP files
- `scripts/publish_to_repo.py`: publishes the addon to `mildman1848.github.io`

## Local Validation

```bash
npm run validate
```
